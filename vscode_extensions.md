
### Extensions

* ESLint

* Path Intellisense
* Reactjs code snippets
* Search node_modules
* vscode-icons

### Useful Shortcuts
|Shortcut|Description|
|---|---|
|⌘B|Toggle Sidebar|
|⌘⇧X|Extension|
|⌘P|Quick Open|
|⌘⇧ [ or ]|Cycle between tabs|
|⌘I|Select current line|
|⌥↓↑|Move Line Up/Down|
|⌘[|Outdent Line|
|⌘]|Indent Line|
|⌘K M|Select Language|
|⌘⇧V|Markdown Preview|
